# Kalkulator Flutter

Aplikasi kalkulator sederhana yang dibangun menggunakan Flutter.

## Fitur
- Penjumlahan
- Pengurangan
- Perkalian
- Pembagian
- Modulus

## Cara Menjalankan
1. Clone repositori ini.
2. Jalankan `flutter pub get` untuk menginstal dependencies.
3. Jalankan `flutter run` untuk menjalankan aplikasi.
